# User activity ETL Pipeline

## Project Overview
This project demonstrates a basic ETL pipeline using Databricks and PySpark for a user activity data.

## Phase 1
- Load JSON data
- Validate schema
- Explore dataset

## Phase 2
- Remove duplicate records
- Store cleaned data in a Spark table
- Check Spark table

## Technologies
- Databricks
- Apache Spark
- PySpark
- JSON
- GitHub
- SQL
